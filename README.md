# BizzyBee

Backend of BizzyBee application using Python and SQLite to manage business data. Businesses can register by providing their name, owner’s name, short description and niche, which are stored in an SQLite database. The database is structured to categorize businesses by niche (ex. “Food”, “Tech”, “Beauty”), allowing users to filter and search for different businesses. The filtering system retrieves and displays matching results from the database, making it easy for users to discover businesses relevant to their needs.
